# Manolo's Vibe AI - Project Guide

## Tech Stack
- **Frontend:** React, Tailwind CSS (Mobile-responsive, Bottom Nav)
- **Backend/Auth:** Supabase
- **Payments:** Stripe
- **Deployment:** Lovable / GitHub sync
- **Aesthetic:** Dark mode only (Nightclub/Social Vibe)

## Core Features
- Social feed & messaging
- Venue discovery & event management
- Talent profiles & payment processing

## Development Standards
- **Style:** Clean, functional components.
- **Performance:** Prioritize optimized Supabase queries.
- **Naming:** Manolo Management branding.

## Common Tasks
- **Build:** npm run dev
- **Vibe Engine:** src/services/vibe-logic.ts
